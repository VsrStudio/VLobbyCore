# VLobbyCore
Replica Venity Network LobbyCore

# Feature
- [X] Vip Chat
- [X] Custom Item
- [X] FriendSystem
- [X] Custom Join Massage
- [X] Games UI
- [X] MultiCount Support
- [X] Simple Protection
- [X] Profile Server Stats/Xbox
- [X] Easy Stup
